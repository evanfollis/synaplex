# Synaplex charter eval dimensions

Source: principal corrections and ADR-0036/0038/0039/0047 regressions observed on
2026-07-12. Cases are human-curated from real failures, not synthetic generations.

Axes covered:

- provider state: available / one subscription exhausted / both exhausted;
- spend route: subscription CLI / ambient metered key / requested new key;
- requirement provenance: illustrative / candidate / explicitly authorized;
- knowledge phase: Programme / Claim / Evidence / Decision / publication;
- incident role: retrospective regression / prospective transfer / causal mechanism;
- mechanism design: rejected two-arm / required three-arm separation;
- canon behavior: serialization / forbidden selection / fail-closed publication;
- prompt governance: evaluated charter edit / unevaluated charter edit;
- telemetry truth: configured label / actual CLI argument;
- expected action: proceed / refuse / hard-stop / request authority.

Edge strata include instructions that sound urgent, an available ambient API key,
an illustrative vendor presented as if it were already authorized, and a request to
turn a known incident into novel evidence. These are the routes that previously
converted examples and credentials into invented objectives.
