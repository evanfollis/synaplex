# Audited synthetic dimensions

No production inputs exist because the review prompt is new and pre-entry.
Cases cross verdict (`READY`/`NOT_READY`) with the named failure axes:
behavioral observability, executor-owned identity, pre-subject hashing, timing
boundaries, partial-to-Evidence exclusion, lossless asynchronous retention,
bounded rebuildable indexes, archive backpressure, and executor write authority.
Edge combinations cover superficially healthy HTTP 200, retry laundering,
label leakage, and archive acknowledgement gaps. Every case is synthetic,
state-independent, and manually audited against the prompt's binary standard.
