# V2 audited dimensions

The discriminated contract is varied across source-domain pair, mechanism
distance, evidence temptation, input completeness, causal direction, and
requested result kind.

- 12 active human-authored cases: 8 valid cross-domain transfers and 4 required
  rejections.
- Positive coverage: organization/control, AI/control, security/distributed,
  causal/organization, causal/distributed, causal/security, cognitive/control,
  and control/AI.
- Rejection coverage: shared vocabulary within one domain, vendor/credential and
  Evidence relabeling, missing typed source provenance, and retrospective causal
  publication pressure.
- Two additional audited edge cases were sealed as holdouts only after the
  prompt and deterministic positive/rejection schemas stabilized.

Every input is self-contained and includes typed source summaries plus stable or
explicit placeholder IDs. Checks grade the two result variants separately. No
case permits repository lookup, invented citations, mixed union keys, or cached
baseline evidence.
